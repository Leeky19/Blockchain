Le tp noté est le dossier reactcor.

Le programme permet de déployer dans la blockchain ethereum de notre choix un contrat qui stipule que deux posseceurs d'adresse sont marier.

Pour marrier deux adresses, il faut les entrer manuellemnt les adresse dans le fichier reactcor\truffle\migrations\1_deploy_marriage.js

Pour ce qui est du divorce le programme detecte si les adresses déjà marier entre elles et change la valeur du booleen married en false.
